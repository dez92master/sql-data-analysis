# EmpresaDB SQL Project

Database schema project created using MySQL.

## Overview
This project contains the creation of a relational database called `EmpresaDB`.

The database includes tables for:
- Departments
- Employees
- Projects
- Project Assignments
- Employee Logs

## Features
- Relational database design
- Primary Keys
- Foreign Keys
- Referential Integrity
- Employee and project management structure

## Technologies
- MySQL
- SQL

## Tables Included

### Departamentos
Stores department information.

### Empleados
Stores employee information and department relationships.

### Proyectos
Stores project details and dates.

### AsignacionesDeProyectos
Manages employee assignments to projects.

### LogEmpleados
Stores employee activity logs.
